1 Errors

1.1 Timeout Error

  • 解决方法: Tools->Server Console, 打开 Server Console面板，Server Gateways->My Computer,右键属性，修改如下两项。

  • 返回Server Gateways->My Computer，右键Test Gateway，测试Gateway是否正常。

1.2 ls_license_vs2008.dll not found

  • 解决方法: 找到文件夹 C:\Program Files (x86)\Common Files\Accelrys\LicensePack （64位） 或 C:\Program Files (x86)\Common Files\Accelrys\LicensePack （32位）, 将其所有的dll文件复制到 C:\Program Files (x86)\Accelrys\Materials Studio 8.0\bin目录下。32位系统类似。
  • 不要使用360清理系统，这个清理比较激进，容易过度清理掉一些库文件。如果想清理的话，推荐CCleaner。

1.3 Win10关键字不在字典中

  1. 可能与Microsoft .NET Framework 的版本更新有关，卸载掉现在电脑上安装的.NET 4.6.X，在微软官网上下载并安装.NET 4.5.2。
  2. 或者安装更新版本的Materials Studio.

1.4 License Error

  1. 控制面板(Win7)/设置(Win10)->搜索UAC->更改用户账户控制->调到最低。

  1. 我的电脑->右键属性->找到计算机名。
  2. 修改msilicense.dat，修改计算机名。

  1. 以管理员身份运行License Administrator->Install License， 找到msilicense.dat的路径，点击install,等待安装完成。

1.5 安装失败且无法卸载问题

  • 可能是安装不完整。对于64位系统，很可能是Accelrys Materials Studio 7.0 (x64 Server).msi和Accelrys Materials Studio Gateway Service (x64).msi未安装完整，将这两个再装一次，然后再全部卸载，再重装。

2 MS Examples

2.1 Dmol Tansport

  1. File->Import,Sturctures-metals.xsd,
  2. Build->Surfaces->Cleave Surfaces, select 1 1 1 for Cleave plane (h k l) , select 12 for Thickness.
  3. Build->Crystals->Build Vacuum Slab. select 7 for Vacuum Thickness,0 for slab position.
  4. Build->Symmetry->Lattice parametres->Advanced, Select “C along Z, B in YZ plane”
  5. Right click->Display Style->Lattice,Select original for Style.
  6. Move 12 layers along Z direction, let 6 layers in cell and 6 layers out of cell.
  7. Build->Symmetry->Redifine Lattice, Select " 1 0 0 0 1 0 0 0 1"
  8. Build->Symmetry->Redifine Lattice, Select " 2 1 0 0 2 0 0 0 1"
  9. Build->Symmetry->Nonperiodic SuperStructure.
  10. Build Transport Device->Build Electrode, Buid for +Z and -Z.
  11. Modify->Constraints, fix the six layers as Electrode.
  12. copy B40, and paste it between two slab (6 layers).
  13. optimize, get energy .vs width. many widths.
