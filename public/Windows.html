1 Windows 缩放问题

  • 在win10中，如果把系统显示缩放调到125%，文字会发生模糊，所以这个缩放应该保持100%。 但可以更改文本和其他项目的大小， 把仅更改文本大小中的选项： 标题栏、菜单、消息框、调色板标题、图标、工具提示等全由9号字提升到12号。360浏览器缩放125%，若标题栏、收藏栏等字体太小，将上面的消息框、图标调至更大，如16.
  • 字体大小也可以在注册表中更改：[HKEY_CURRENT_USER\Control Panel\Desktop\WindowMetrics],将CaptionFont，IconFont，MenuFont，MessageFont，SmCaptionFont以及StatusFont中的0000后面大F4跳为F0(F4 = 9,F3 = 10,F1 = 11,F0 = 12,EF = 13,ED = 14,EC = 15,E0 = 24)。

5 视力保护色（豆沙色）：

HSU 设置为： 色度(HUE) 85 饱和度(SAT) 123 亮度 205
三原色RGB设置为：红199 绿237 蓝204
RGB十六进制： 红C7 绿ED 蓝CC
淡绿色（豆沙色）：#C7EDCC

5.1 CMD windows 颜色

  • 背景色RGB： 199|237|204
  • 文字颜色RGB： 136|23|152

5.2 屏幕设定：

5.2.1 win7系统下

  1. 桌面点鼠标右键，选择 个性化 - 窗口颜色 - 高级外观设置
  2. 项目栏选“窗口” - 再点后面的“颜色” - “其他”，按上面XP设置方式一样进行设置就是了。 OK，现在我们的窗口颜色就是极具保护视力的嫩绿色了。

5.2.2 win 10：

  1. 快捷键Win+R打开运行，输入regedit运行注册表编辑程序；
  2. 依次打开[HKEY_CURRENT_USER\Control Panel\Colors]，双击修改Window字符串值为“204 232 207”（注意空格），此为豆沙绿护眼色，需要修改为别的颜色可自行输入颜色RGB值 199 237 204
  3. 关闭注册表，注销或重启系统即可生效。

5.2.3 Sumatra PDF：

设置->高级选项： UseSysColors = false 改为 true

5.2.4 Foxit pdf：

Edit->preference->Accessibility->选择Replace document colors->custom background

5.2.5 Firefox:

设置->颜色->选择使用系统颜色，高对比度时覆盖。

6 文件夹或目录背景右键出现无效的cmd

  • 打开注册表regedit，找到HKEY_CLASSES_ROOT\Directory\Background\shell或HKEY_CLASSES_ROOT\Directory\Background\shell下，查看是否有多余的cmd，右键这个cmd项打开权限卡->高级，所有者改为everyone，其权限改为完全控制，勾选下面可从此权限几种的项目替代所有子对象权限项目，确定。右键点击cmd项删除。

7 Windows 10之 隐藏“此电脑”窗口的6个额外文件夹及移除OneDrive

转自： http://ycrsjxy.blog.51cto.com/618627/1729092

默认情况下，Windows 10的“此电脑”窗口会显示“视频、图片、文档、下载、音乐、桌面“六个文件夹。大部分情况下，用户使用这几个文件夹的机会并不多，有时会觉得有些“碍眼“。那么，用户是否能删除（隐藏）他们呢？

要隐藏这几个文件，用户只需要在管理员权限下运行regedit程序，修改相应键值即可。

（具体键值可以从附件下载）

将相应项下的“ThisPCPolicy”修改为“Hide”。

桌面文件夹对应的PropertyBag项内可能没有“ThisPCPolicy”，手动新建一个“字符串值”将其命名为ThisPCPolicy，并赋予Hide值同样可以起到隐藏的作用。

Windows Registry Editor Version 5.00

[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\FolderDescriptions\{B4BFCC3A-DB2C-424C-B029-7FE99A87C641}\PropertyBag]

"ThisPCPolicy"="Hide"

[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\FolderDescriptions\{a0c69a99-21c8-4671-8703-7934162fcf1d}\PropertyBag]

"ThisPCPolicy"="Hide"

[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\FolderDescriptions\{7d83ee9b-2244-4e70-b1f5-5393042af1e4}\PropertyBag]

"ThisPCPolicy"="Hide"

[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\FolderDescriptions\{f42ee2d3-909f-4907-8871-4c22fc0bf756}\PropertyBag]

"ThisPCPolicy"="Hide"

[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\FolderDescriptions\{0ddd015d-b06c-45d5-8c4c-f59713854639}\PropertyBag]

"ThisPCPolicy"="Hide"

[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\FolderDescriptions\{35286a68-3c57-41a1-bbb1-0eae73d76c95}\PropertyBag]

"ThisPCPolicy"="Hide"

至Win8系统开始微软在系统中集成了OneDrive云服务功能，最初这项功能使用起来还是很方便的，不过由于某些原因在使用了长达两年的OneDrive后最终还是选择放弃了。

在Win10中，OneDrive默认会出现在资源管理器左侧导航窗格中，即使未登录OneDrive也会出现。那么既然已不使用它，就想着把它去除掉。

移除或隐藏OneDrive图标的方法有两个，一个是注册表修改隐藏，还有一个就是彻底关闭OneDrive功能，具体操作如下。

方法一：禁用OneDrive

1、在win10系统下按下WIN R 组合键，然后输入 gpedit.msc 回车；

2、打开本地组策略后，我们依次展开：“计算机配置”——“管理模板”——“Windows组件”——“OneDrive”；

3、双击右侧“禁止使用 Onedrive 进行文件储存“选项，设置为启用，如下图，然后确定即可。

4、最后需要重启一下计算机才可以。

方法二：注册表修改

1、按Win R打开运行，输入regedit回车打开注册表编辑器；

2、导航至HKEY_CLASSES_ROOT{018D5C66-4533-4307-9B53-224DE2ED1FE6}

3、在右侧找到Attributes，双击将其值修改为f090004d。

如果你遇到注册表权限问题可参考 Win10系统修改注册表需要权限如何获得？

OK，通过上述两个方法都可以不显示资源管理器导航栏OneDrive图标，一个是只隐藏，而一个是需要关闭OneDrive.
