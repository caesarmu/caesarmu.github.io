  • 安装WPS，将安装目录下office6、utility、wtoolex复制到D:\Datas\Softs\DraftTools\dosbin\WPSOffice下，

3.3 Latex for WPS

3.3.1 problem for TMP

entering extended mode
! I can’t find file `c:/users/admini’.
?
<*> c:/users/admini~
1/appdata/local/temp/inkscape-wawis7/eq.tex
Please type another input file name:

Answer: It is a problem with the TMP variable. It was C:\Users\Administrator\AppData\Local\Temp,
but it was abbrivated to be C:\Users\Admini~1\AppData\Local\Temp, which was not recognized by Pdflatex.
so mkdir D:\Temp\systmp, and change TMP variable to D:\Temp\systmp in environment setting.

3.3.2 Using Latex in WPS

  1. 安装WPS office(安装在C:\Program Files (x86)而非默认的C:\Users下)和MikTex。
  2. 下载LaTeXEE1.01, https://sourceforge.net/projects/latexee/.
  3. 解压并以管理员身份运行latexeqedit.exe，打开Setting设置texify.exe和dvipng.exe的路径，然后选择options-save。测试Equation，确保能正常运行。
  4. 用WPS打开ppt， 插入-对象-Latex Formula，确定。latexeqedit.exe自动启动，输入公式，预览后关闭窗口，在幻灯片上就生成了png格式的公式。可以双击图片修改公式。也可以关闭窗口前点击File-Save as， 保存为ltx文件，以后用latexeqedit.exe打开此文件，自动生成公式。
  5. 用WPS打开Word,操作类似。
  6. 据说此方法还支持winword, wordpad, OpenOffice等等。
  7. Preamble
\usepackage{amsmath}
\usepackage{amssymb}
\usepackage{color} 
\everymath{\color{red}}
\everydisplay{\color{red}}
\usepackage{amsmath} 
\renewcommand{\vec}[1]{\boldsymbol{#1}} % Uncomment for BOLD vectors
%Multiline: use math environment and add \\ \indent

3.3.2.1 OLE Failed on Win10

  • 以管理员身份运行LaTeXEE1.01，并保存一次。(不行重启)
  • 还不行，运行risk.reg,取消一些文件的签名要求，重启。
  • 还不行，将 C:\Users\Administrator\AppData\Local\Temp里的LatexEE相关的东西删除一次，
  • 还不行，WPS配置工具，修复一次，还不行重新注册组件，重启。或以管理员运行writer一次。
  • 还不行，重装office。

3.4 WPS国际版

  • 安装WPS个人版，将其中xxxx\office6\mui复制出来备用，卸载WPS个人版。

  • 安装WPS国际版

  • 找到国际版的安装目录,删除xxxx\office6\mui，复制个人版的xxxx\office6\mui到此处。

  • 重命名xxxx6下的wpscenter.exe，wpscloudlaunch.exe， wpscloudsvr.exe和wpsupdate.exe等文件。

  • 关闭自动更新。

3.5 WPS共享词典

  1. 资源管理器下进入目录%APPDATA%\kingsoft\office6\en_US，将CUSTOM.DIC复制到云共享目录下
  2. 文件⇒选项⇒拼写检查⇒自定义词典⇒添加刚才复制的词典(云共享目录下)，并设其为默认，删除原有的默认词典。
  3. 在其它主机进行步骤2的操作。

3.6 word中图片高分辨转为pdf

  1. wps：文件->选项->常规与保存：勾选不压缩文件中的图像。
  2. 导入本地图片。
  3. 生成pdf时不使用wps自带的“Export to pdf,”而是选择：文件->打印->Microsoft Print to PDF。

3.7 修改WPS模板

3.7.1 建立模板

  1. 打开docx文件，修改相应的模板
  2. 分别以dotx和dotm格式将其保存在%DosHome%\InWords\software\wps\model下(mymodel.dotx/Normal.dotm)

3.7.2 临时修改

  1. 打开docx文件，文件⇒新建⇒本机上的模板⇒，选择mymodel.dotx,并设为默认模板，保存并关闭。
  2. 以后打开docx文件，文件⇒新建⇒从默认模板新建。

3.7.3 永久修改

  • 用Normal.dotm替换%AppData%\kingsoft\office6\templates\wps\zh_CN\Normal.dotm
