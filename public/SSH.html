1 xshell and Xming

生成SSH.bat,

@echo off
start /B C:\"Program Files (x86)"\NetSarang\"Xshell 5"\Xshell.exe"
start /B C:\"Program Files (x86)"\Xming\Xming.exe :0 -clipboard -multiwindow

生成快捷方式，并更换图标。

2 To Install MobaXterm

  • unzip MobaXterm_v8.6.zip
  • copy MobaXterm_v8.6/MobaXterm_Personal_8.6.exe to E:\Softs\DoNotSync\MobaXterm
  • double click MobaXterm_Personal_8.6.exe
  • copy Configuration file to E:\Softs\DoNotSync\MobaXterm or C:\Users\Administrator\Documents\MobaXterm
  • double click packages, install mintty.exe, rsync.exe, expect.exe,gawk.exe
  • double click MobaXterm.e.reg
  • copy .bashrc to E:\Softs\DoNotSync\MobaXterm\Mobahome

3 To install PuTTY and WinSCP

3.1 Install PuTTY

to: C:\Program Files (x86)\WinSCP\PuTTY

3.1.1 Change Setting for PuTTY

  • 选择Default setting, load, 选择window-Apprearance- font setting -change,
    Bitstream Vera Sans Mono，常规，12.
  • Connection->SSH->x11，勾上Enable X11 forwarding。

3.2 Install WinSCP

default at: C:\Program Files (x86)\WinSCP

3.2.1 Change Setting for WinSCP

打开WinSCP-工具-选项-集成-应用程序：
PuTTY客户端路径： %PROGRAMFILES(X86)%\WinSCP\PuTTY\putty.exe -t -m "%TEMP%\putty.txt" !cmd.exe /c echo cd "!/" ; /bin/bash -login > "%TEMP%\putty.txt"
勾选 “记住密码并传递给PuTTY”
